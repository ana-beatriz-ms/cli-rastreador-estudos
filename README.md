# Rastreador de estudos - timer para foco

Um rastreador de estudos simples e eficiente para terminal, desenvolvido com Node.js. 

## Funcionalidades

- Timer de estudos
- Estatísticas de tempo de estudo
- Historico persistente
- Organização por matéria

## Como usar

### Pré-requisito
- Node.js instalado no seu computador

### Instalação

1. Clone o repositório:

git clone https://github.com/seu-usuario/rastreador-estudos.git

2. Entre no diretório do projeto:

cd rastreador-estudos

3. Execute o programa:

node index.js

## Como funciona
Ao executar o programa, você verá um meno com duas opções:
- Iniciar sessão de foco: escolha matéria e tempo de foco, depois veja o cronometro no terminal.
- Ver estatísticas de estudo: confira seu tempo de histórico de minutos estudos diário, semanal, e total. 

## Tecnologias usadas
- Node.js: Ambiente de execução JavaScript
- File System (fs): Manipulação de arquivos JSON
- Readline: Interface de linha de comando interativa
